# Slide Snapshot to ChatGPT

Tiện ích mở rộng Chrome nhỏ giúp bạn *crop* nhanh vùng khung trình chiếu (Google Slides, PowerPoint web, v.v.) rồi tự động:

1. Sao chép ảnh đã crop vào clipboard.
2. Mở tab ChatGPT để bạn dán trực tiếp (Ctrl / Cmd + V).

Không cần Snipping Tool hay tải ảnh thủ công nữa.

---

## Cài đặt (chạy ở chế độ Developer)

1. Mở `chrome://extensions`.
2. Bật **Developer mode** (góc trên bên phải).
3. Chọn **Load unpacked** rồi trỏ tới thư mục `slide-to-chatgpt`.
4. Ghim biểu tượng tiện ích lên thanh công cụ nếu muốn.

---

## Cách dùng

1. Vào trang trình chiếu (ví dụ: Google Slides, PowerPoint web, Canva...).
2. Bấm vào biểu tượng tiện ích **Slide Snapshot to ChatGPT** hoặc dùng nút máy ảnh nổi ở góc phải dưới (có thể tắt/bật trong Options).
3. Tiện ích sẽ ưu tiên tự nhận diện khung slide trung tâm và chụp luôn; nếu không được sẽ hiện overlay để bạn kéo chuột chọn vùng.
4. Sau khi chụp xong, tiện ích sẽ:

   * Sao chép ảnh vào clipboard (dự phòng).
   * Bạn hãy mở tab ChatGPT, ctrl + v.
5. Bạn chỉ cần bổ sung câu hỏi (nếu muốn) rồi nhấn Enter.
